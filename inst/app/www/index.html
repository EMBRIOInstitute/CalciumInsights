Calcium transient analysis platform

CalciumInsights

A compact and interactive Shiny application for calcium transient analysis using two complementary workflows: FFT-based peak detection and wavelet ridgewalking.

CalciumInsights helps users load calcium imaging traces, select a region of interest, define baseline strategies, detect transient events, evaluate baseline sensitivity, and extract quantitative metrics such as amplitude, timing, frequency, FWHM, prominence, and area under the curve.

Main Analysis Modules

FFT + Baseline Analysis

Peak-based analysis workflow for denoising, baseline selection, peak detection, and transient metric extraction.

  • FFT low-pass smoothing
  • Multiple baseline definitions
  • Peak detection and transient metrics
  • Baseline sensitivity analysis

Wavelet Ridgewalking

Multiscale event-detection workflow designed to identify calcium events as ridge-like structures in a wavelet transform map.

  • Continuous wavelet transform map
  • Ridge-based event detection
  • Biological post-filtering
  • Wavelet-derived metrics and baseline sensitivity

Why two detection methods?

FFT-based peak detection is intuitive for clear transient peaks, while wavelet ridgewalking can help detect broader or lower-amplitude events that may appear across multiple temporal scales.

Why baseline sensitivity matters

Amplitude, area, and duration-related metrics can change depending on the baseline definition. CalciumInsights makes this sensitivity visible before interpreting biological conclusions.
